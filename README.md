# Инструкция по загрузке сайта на InfinityFree

## Что у вас есть:
- ✅ Готовый лендинг (index.html, styles.css, script.js)
- ✅ Страница политики конфиденциальности (privacy.html)
- ✅ Адаптивный дизайн для мобильных устройств

## Шаг 1: Подготовка файлов для загрузки

### Вариант А: Загрузка через FileZilla (FTP)
1. **Скачайте FileZilla Client:** https://filezilla-project.org/
2. **Подключитесь к FTP:**
   - **Хост:** ftp.body-fix.rf.gd (или адрес из панели InfinityFree)
   - **Имя пользователя:** из панели InfinityFree
   - **Пароль:** из панели InfinityFree
   - **Порт:** 21

3. **Загрузите файлы:**
   - Перейдите в папку `htdocs` на сервере
   - Загрузите ВСЕ файлы из папки `landing`:
     - index.html
     - styles.css
     - script.js
     - privacy.html

### Вариант Б: Загрузка через панель InfinityFree
1. Войдите в панель управления InfinityFree
2. Перейдите в **File Manager**
3. Откройте папку `htdocs`
4. Загрузите файлы через кнопку **Upload**

## Шаг 2: Настройка ссылок на мессенджеры

### ВАЖНО! Перед загрузкой нужно заменить ссылки:

В файле `index.html` найдите и замените:

1. **WhatsApp ссылки (3 места):**
   ```
   https://wa.me/ВАШНОМЕР?text=Здравствуйте,%20Лада!%20Хочу%20бесплатный%20разбор
   ```
   Замените `ВАШНОМЕР` на реальный номер WhatsApp в формате: `79001234567` (без + и пробелов)

2. **Telegram ссылка:**
   ```
   https://t.me/ВАШНИК?text=Здравствуйте,%20Лада!%20Хочу%20бесплатный%20разбор
   ```
   Замените `ВАШНИК` на ваш username в Telegram (без @)

### Как заменить:
1. Откройте `index.html` в текстовом редакторе
2. Нажмите `Ctrl + H` (Замена)
3. Найдите `ВАШНОМЕР` и замените на ваш номер
4. Найдите `ВАШНИК` и замените на ваш Telegram username
5. Сохраните файл

## Шаг 3: Добавление изображений (опционально)

### Если хотите добавить фото на сайт:

1. **Подготовьте фото:**
   - Ваше фото для Hero-секции
   - Фото до/после (если есть)
   - Отзывы/скриншоты (если есть)

2. **Создайте папку `images`** в папке `landing`

3. **Добавьте фото** в папку `images`

4. **Обновите CSS** в файле `styles.css`:
   ```css
   .hero {
       background: linear-gradient(135deg, #F5F7FA 0%, #E8F5E9 100%) 
                   url('../images/your-photo.jpg') center/cover;
       /* Добавьте это, если хотите фоновое фото */
   }
   ```

## Шаг 4: Проверка сайта

После загрузки:
1. Откройте `http://body-fix.rf.gd`
2. Проверьте все ссылки и кнопки
3. Протестируйте на мобильном устройстве
4. Убедитесь, что ссылки на WhatsApp/Telegram работают

## Шаг 5: Настройка SSL (HTTPS)

InfinityFree предоставляет бесплатный SSL:
1. Войдите в панель InfinityFree
2. Найдите раздел **SSL** или **HTTPS**
3. Активируйте бесплатный SSL сертификат
4. Сайт будет доступен по `https://body-fix.rf.gd`
